# Space Explorer 🌌

## Description
An interactive web app to explore planets, filter by type, and save favorites.

## Features
- 🌑 View planet details when clicked
- 🏷️ Filter planets by type
- ⭐ Add/remove favorites
- 🌗 Toggle dark mode
